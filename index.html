How It Works

1

Download and unzip

Download the toolkit. Unzip it anywhere on your computer. No install needed.

2

Run START

Windows: Double-click START_HERE_WINDOWS
Mac: Open Terminal, type bash (with a space), then drag START_HERE_MAC into the Terminal window and press Enter.
This installs the required packages. Only needed the first time.

3

Export your data

Export your work items from your ALM tool (Jira, Rally, Azure DevOps, or any other tool) as a CSV or Excel file.

4

Drop the file and confirm columns

Put the exported file in the input folder. The toolkit reads your columns and matches them automatically. Just press Enter to confirm. This only happens once — it remembers your choice for next time.

5

See your reports

Your dashboard opens in the browser with all your metrics. Next time, just drop a new file — no setup needed, reports update automatically.

Reports You Get

The toolkit generates reports based on what is in your data. More columns = more reports.

Velocity

Scope Change

How much work was added or removed after commitment. Track scope creep across sprints.

Cycle Time

How long items take from creation to completion. Average, median, and distribution by team.

Epic / Feature Progress

Completion status of each epic or feature. See which ones are on track and which are behind.

Work Distribution

Breakdown of items by status, type, and priority. Understand where work is stuck.

Team Comparison

Side-by-side view of all teams. Points delivered, completion rates, and average item size.

Common Questions

Which ALM tools does this work with?

Any tool that can export data as CSV or Excel. This includes Jira, Rally, Azure DevOps, VersionOne, Targetprocess, and many others. The toolkit reads your column names and maps them automatically.

Do I need to know Python?

No. You just double-click START_HERE and drop files into a folder. Python is needed on your computer, but you do not need to write any code.

What if my columns have different names?

The toolkit automatically matches your columns to the fields it needs. For example, "Story Points", "SP", "Estimate", and "Points" all map to the same thing. If the auto-match is wrong, you can fix it — and it remembers your choice.

Does this work on Windows and Mac?

Yes. On Windows, double-click START_HERE_WINDOWS. On Mac, open Terminal and run the START_HERE_MAC file (drag it into Terminal and press Enter). Both do the same thing.

Mac shows a security warning. What do I do?

Instead of double-clicking, open the Terminal app (search for "Terminal" in Spotlight), type bash followed by a space, then drag the START_HERE_MAC file into the Terminal window and press Enter. This bypasses the warning safely.

Can I use this at work?

This toolkit is provided as a free learning and personal productivity tool. Check with your organization's policies before using it with work data. The code is open source so your team can review it.

How do I install Python?

Go to python.org/downloads and download Python 3. On Windows, make sure to check "Add Python to PATH" during installation. On Mac, Python 3 may already be installed.
